# Siamese Networks

## 基本信息

- **代表性论文**: 
  - [Signature Verification using a "Siamese" Time Delay Neural Network](https://papers.nips.cc/paper_files/paper/1993/hash/288cc0ff022877bd3df94bc9360b9c5d-Abstract.html) (NIPS 1993)
  - [Siamese Neural Networks for One-shot Image Recognition](https://www.cs.cmu.edu/~rsalakhu/papers/oneshot1.pdf) (ICML 2015)
- **作者**: Jane Bromley et al. / Gregory Koch et al.

## 创新点

1. **参数共享的双塔网络**: 两个网络共享权重，分别处理两个输入
2. **度量学习**: 学习输入对之间的相似度而非分类
3. **单样本学习**: 仅需一个样本即可识别新类别

## 核心原理

### 架构

1. 两个输入分别通过**共享权重**的编码器
2. 编码器输出两个嵌入向量
3. 计算嵌入向量的距离（L1/L2 或余弦相似度）
4. 损失函数：对比损失（正例拉近，负例推远）

### 对比损失

$$\\mathcal{L} = y \\cdot d^2 + (1-y) \\cdot \\max(0, m-d)^2$$

- $d$: 两个嵌入的欧氏距离
- $m$: margin（通常 1.0）
- $y$: 1（正例对）/ 0（负例对）

### 与现代方法的关系

Siamese Networks 是现代对比学习的先驱：
- Siamese → SimCLR / MoCo（增加了数据增强和投影头）
- Siamese → 非对比方法（BYOL、SimSiam，加了不对称设计）

## 训练方法

### 核心思想：给网络两张照片——告诉它"这两个是同一个人"或"这两个是不同人"。网络自己学会提取哪些特征来判断"相同 vs 不同"。这就是 Siamese 网络——它不学习"这是谁"，而是学习"这两个像不像"

Siamese Networks（孪生网络）是度量学习和对比学习的精神先驱。它的设计极其优雅：两个输入 → 两个**共享权重**的编码器 → 两个嵌入向量 → 计算距离 → 对比损失。三十年来，这个范式支撑了人脸识别、签名验证、单样本学习等无数应用。

> Siamese Network = 双编码器（参数共享）+ 距离度量（欧氏/余弦）+ 对比损失（正例拉近、负例推远到 margin 以外）。这是现代自监督学习（SimCLR、MoCo、BYOL）的概念源头。

### 训练流水线（Step by Step）

#### Step 1 — 双塔架构（参数共享）

```text
输入 1 → 编码器 → 嵌入向量 1
输入 2 → 编码器 → 嵌入向量 2
         ↑
    完全相同的权重（孪生 = 共享）

相似度 = 1 - Distance(嵌入向量1, 嵌入向量2)  或 Cosine(嵌入1, 嵌入2)
```

| 组件 | 早期实现 (1993) | 深度学习实现 (2015) |
|------|---------------|----------------|
| 编码器 | 时延神经网络 (TDNN) | CNN (VGG/ResNet) |
| 输入类型 | 签名图片（二值） | 灰度图像 (Omniglot, MNIST) |
| 嵌入维度 | ~50 | 256-4096 |

#### Step 2 — 对比损失（Contrastive Loss）

核心损失函数——Siamese Network 最重要的设计元素：

$$\mathcal{L}(y, d) = y \cdot d^2 + (1-y) \cdot \max(0, m - d)^2$$

| 项 | 含义 | 效果 |
|----|------|------|
| $d = \|f(A) - f(B)\|_2$ | 两个嵌入的欧氏距离 | — |
| $y = 1$ (正例) | A 和 B 是同一类/同一人 | 惩罚 $d^2$ → **拉近** |
| $y = 0$ (负例) | A 和 B 是不同类/不同人 | 惩罚 $\max(0, m-d)^2$ → **推远到 margin m** |
| $m$ (margin) | 期望的距离间隔 | 通常 1.0-2.0 |

**损失函数的直观理解**：

```text
正例对 (同一个人):
  d 越小越好 → 损失 = d²
  如果 d=0.1 → 损失很小

负例对 (不同人):
  d 越大越好 → 至少大于 m
  如果 d=2.5, m=2.0 → 损失 = 0（完美！负例足够远）
  如果 d=0.8, m=2.0 → 损失 = (2.0-0.8)² = 1.44（太近！需要推远）
```

#### Step 3 — 成对/三元组训练

**成对训练（Pair-based）**：

```text
训练集: 同一个人/不同人的配对

正例对:
  (张三_照片1, 张三_照片2, y=1)
  (猫_A_图1, 猫_A_图2, y=1)

负例对:
  (张三_照片1, 李四_照片1, y=0)
  (猫_A_图1, 狗_B_图1, y=0)

训练: 交替采样正例对和负例对
```

**三元组训练（Triplet Loss）—— 后续改进**：

$$\mathcal{L}_{\text{triplet}} = \max(0, d(A, P) - d(A, N) + m)$$

- A: Anchor（锚样本）
- P: Positive（同类的另一个样本）
- N: Negative（不同类的样本）

三元组的思想：不光要求"同类的近"，还要求"比不同类的更近"——相对距离约束。

#### Step 4 — 完整训练流程

```text
对于每个 epoch:
  采样一批正例对和负例对

  对于每对 (A, B):
    嵌入 A = Encoder(A)
    嵌入 B = Encoder(B)
    d = ||嵌入A - 嵌入B||₂
  
  计算对比损失 L(y, d)
  反向传播 → 更新 Encoder 的共享权重
```

| 参数 | 早期 Siamese | 现代 Siamese |
|------|------------|------------|
| 数据集 | Omniglot (1623 类) | 大规模人脸数据集 |
| 正负比 | 1:1 | 1:1 到 1:10 |
| 嵌入维度 | 256 | 128-512 |
| Margin $m$ | 1.0 | 0.5-2.0 |
| 优化器 | SGD | Adam/SGD |
| 学习率 | 0.001 | 0.001-0.01 |

#### Step 5 — Siamese → 现代对比学习（演进路线）

Siamese Network 的核心理念演化了现代自监督学习：

```text
Siamese (1993/2015)
  → 双编码器 + 对比损失
  → 需要标签（哪些是同类/不同类）
  ↓
SimCLR (2020)
  → 双编码器 + 数据增强（同一图片的两个增强 = 正例）
  → 无需标签！（增强 = "同类"）
  ↓
MoCo (2020)
  → 双编码器 + 动量更新 + 队列
  → 更大的负例集
  ↓
BYOL / SimSiam (2020/2021)
  → 非对称双编码器 + 无负例
  → 对比学习的终极演化
```

### Siamese Networks vs 现代对比学习

| 维度 | Siamese Networks | SimCLR |
|------|-----------------|--------|
| 标签需求 | **需要（哪些是同一个人）** | **无需（数据增强=同一个人）** |
| 正例定义 | 人工标注的同类 | 同一图片的两种增强 |
| 负例 | 人工标注的不同类 | Mini-batch 中其他图片 |
| 编码器 | 共享权重 | 共享权重 |
| 投影头 | 无（直接度量） | **有（MLP 投影）** |
| 规模 | 小 | 大（4096+ batch） |

### 训练的实用价值

1. **度量学习的范式确立**：不学分类器，学"相似度函数" → 开集问题（新类别无需重训练）
2. **人脸识别的基石**：FaceNet、ArcFace、CosFace 都从 Siamese 思想出发
3. **少样本学习**：仅需一个样本就能判断"是否同类" → Omniglot 上的 1-shot 精度 ~92%
4. **现代 SSL 的精神先驱**：SimCLR、MoCo、BYOL 都可以被视为 Siamese 的"自监督版本"
5. **签名/指纹/虹膜验证**：生物特征验证的工业标准架构
