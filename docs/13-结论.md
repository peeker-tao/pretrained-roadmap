# 第 13 章：结论

## 13.1 核心回顾

本文按**预训练目标函数**这一核心维度，系统梳理了从监督预训练到基础模型时代的预训练方法演进。

### 三条主线

1. **监督信号从强到弱**
   - 人工标签 → 数据自身结构 → 弱监督/跨模态 → 无监督
   - 信号的强度和质量逐渐放宽，但数据规模急剧扩大

2. **模态从单一到多元**
   - CV 独自发展 → NLP 独自发展 → 视觉-语言融合 → 多模态原生

3. **能力从特定到通用**
   - 任务特定预训练 → 多任务迁移 → 零样本/少样本 → 涌现能力

### 关键发现

- **不同目标函数产生的表征性质不同**，决定了下游任务的适配策略：
  - 对比方法更适合线性探测（全局特征好）
  - 生成方法更适合全量微调（局部特征好）
  - 多模态对比更适合零样本迁移

- **方法之间的界限正在模糊**：
  - 对比 + 生成（[iBOT](../models/iBOT.md), [data2vec](../models/data2vec.md)）
  - 对比 + 协方差（[VICReg](../models/VICReg.md)）
  - 自回归 + 掩码（[XLNet](../models/XLNet.md)）
  - 多任务联合训练（[CoCa](../models/CoCa.md), [Florence](../models/Florence.md)）

- **规模带来质变**：
  - Scaling Law 揭示了模型、数据、计算量的幂律关系
  - 涌现能力在特定规模阈值后出现
  - 但 [Chinchilla](../models/Chinchilla.md) Scaling 提示我们"不是越大越好"，数据与参数的比例更关键

## 13.2 实用建议

### 根据场景选择预训练方法

| 应用场景 | 推荐方法 | 理由 |
|---------|---------|------|
| 通用视觉分类 | [DINOv2](../models/DINOv2.md), [MAE](../models/MAE.md) | 表征质量高，迁移性好 |
| 稠密预测（检测/分割） | [MAE](../models/MAE.md), [DINO](../models/DINO.md) | 局部特征保持好 |
| 零样本分类 | [CLIP](../models/CLIP.md), [SigLIP](../models/SigLIP.md) | 跨模态对齐，零样本能力强 |
| 视觉-语言理解 | [BLIP-2](../models/BLIP2.md), [LLaVA](../models/LLaVA.md) | 图文理解与生成 |
| 文本理解 | [BERT](../models/BERT.md), [RoBERTa](../models/RoBERTa.md) | 双向上下文 |
| 文本生成 | [GPT](../models/GPT.md), [LLaMA](../models/LLaMA.md) | 自回归，生成质量高 |
| 数据极少 | [SimCLR](../models/SimCLR.md), [BYOL](../models/BYOL.md) | 自监督，无需标注 |
| 大模型微调 | [LoRA](../models/LoRA.md), [QLoRA](../models/QLoRA.md) | 参数高效，成本低 |

### 技术要素的选择原则

- **数据优先**：好的数据 > 好的模型
- **目标函数匹配**：选择与下游任务匹配的预训练目标
- **计算预算**：根据可用资源决定模型规模和训练策略
- **持续跟踪**：预训练方法演进迅速，保持对新方法的关注

## 13.3 终极目标

> **预训练的终极目标是：通用的、可迁移的、安全的智能表征。**

- **通用的**：适用于多种模态、多种任务、多种领域
- **可迁移的**：从大规模数据中学到的知识可以高效迁移到新场景
- **安全的**：表征本身不包含有害、歧视或隐私敏感的信息

这是一个远未完成的目标，但过去十年的进展让我们看到了实现的希望。
