# AlexNet

## 基本信息

- **论文**: [ImageNet Classification with Deep Convolutional Neural Networks](https://papers.nips.cc/paper_files/paper/2012/hash/c399862d3b9d6b76c8436e924a68c45b-Abstract.html)
- **作者**: Alex Krizhevsky, Ilya Sutskever, Geoffrey E. Hinton
- **发表**: NeurIPS 2012

## 创新点

1. **深度 CNN 的大规模成功**: 首个在 ImageNet 上大幅超越传统方法的深度卷积神经网络（Top-5 错误率 15.3%）
2. **ReLU 激活函数**: 使用 Rectified Linear Unit 替代 Sigmoid/Tanh，解决了梯度消失问题并加速训练
3. **Dropout 正则化**: 随机丢弃神经元防止过拟合
4. **GPU 并行训练**: 利用两块 GTX 580 GPU 实现模型并行
5. **局部响应归一化 (LRN)**: 增强神经元对较大响应的竞争

## 核心原理

AlexNet 包含 8 层可学习参数（5 个卷积层 + 3 个全连接层）。输入为 224×224×3 的 RGB 图像，使用 11×11、5×5、3×3 的卷积核提取多尺度特征。最终通过 1000 路的 Softmax 输出分类概率。

## 预训练方法

### 核心思想：用大规模监督数据 + 深度 CNN "暴力"学习视觉特征

在 2012 年之前，计算机视觉的主流是手工设计特征（SIFT、HOG）+ 浅层分类器（SVM）。AlexNet 的革命性在于：**让神经网络直接从百万级标注图像中学习所有特征**——不需要人工设计特征，不需要先验知识，只需要足够的数据、足够的算力和正确的训练技巧。

> AlexNet = ImageNet (120万张图) + 2×GPU 并行 + 8层卷积/全连接 + ReLU/Dropout。它的预训练本质是"监督预训练"——在有标签的大数据集上训练一个分类器，分类任务学到的特征可以迁移到其他视觉任务。

### 训练流水线（Step by Step）

#### Step 1 — 数据准备与增强

ImageNet ILSVRC 2012 包含约 120 万张训练图片，分布在 1000 个类别中。

| 数据增强 | 方法 | 作用 |
|---------|------|------|
| 随机裁剪 | 从 256×256 图像中随机裁 224×224 | 增加平移不变性 |
| 水平翻转 | 50% 概率左右翻转 | 增加对称不变性 |
| PCA 颜色扰动 | 对 RGB 通道应用 PCA，扰动主成分的强度 | 增加颜色不变性 |

**PCA 颜色扰动**是 AlexNet 特有的增强：
```text
1. 对训练集 RGB 像素的 PCA 主成分
2. 添加扰动: [p1,p2,p3] × [α1λ1, α2λ2, α3λ3]
   (αi ~ N(0, 0.1), λi 是特征值)
```
这迫使模型不能依赖特定颜色来分类——光照不变性的关键。

#### Step 2 — 前向传播架构

```text
输入 224×224×3
  ↓ 11×11 Conv (96, stride 4) → ReLU → LRN → MaxPool
  ↓ 5×5 Conv (256) → ReLU → LRN → MaxPool
  ↓ 3×3 Conv (384) → ReLU
  ↓ 3×3 Conv (384) → ReLU
  ↓ 3×3 Conv (256) → ReLU → MaxPool
  ↓ FC 4096 → ReLU → Dropout(0.5)
  ↓ FC 4096 → ReLU → Dropout(0.5)
  ↓ FC 1000 → Softmax
```

**ReLU vs Sigmoid/Tanh**：
$$\text{ReLU}(x) = \max(0, x)$$

| 激活函数 | 梯度 | 训练速度 |
|---------|------|---------|
| Sigmoid | 最大 $0.25$，且 $x$ 大会饱和 | 慢，易梯度消失 |
| **ReLU** | 恒为 $1$（对正输入） | **快 6×**，不饱和 |

> ReLU 的计算极端简单（if x>0: pass, else: 0），在反向传播中梯度不衰减——这是深度网络能被训练的关键要素之一。

#### Step 3 — GPU 模型并行

两块 GTX 580（3GB 显存各）无法装下整个 AlexNet。解决方案：

- **层内并行**：将卷积核分成两组，分别放在两块 GPU 上
- 某些层（Conv3, FC 层）跨 GPU 通信，其余层只在各自 GPU 内通信
- 这种"受限的并行"迫使模型学习互补特征——每块 GPU 的卷积核学会了检测不同的模式

#### Step 4 — 损失函数与优化

**损失函数**：多类交叉熵

$$\mathcal{L} = -\sum_{i=1}^{1000} y_i \log(\hat{y}_i)$$

其中 $y_i$ 是 one-hot 标签，$\hat{y}_i$ 是 softmax 预测概率。

**优化器**：SGD + Momentum

$$v_{t+1} = \mu v_t - \eta \nabla L(\theta_t)$$
$$\theta_{t+1} = \theta_t + v_{t+1}$$

其中 $\mu = 0.9$（动量系数），加速收敛并平滑梯度方向。

#### Step 5 — 正则化策略

| 正则化 | 方法 | 为什么有效 |
|--------|------|-----------|
| **Dropout (0.5)** | 训练时随机将 50% 神经元输出置零 | 防止神经元"偷懒"（依赖特定同伴） |
| **Weight Decay (0.0005)** | $L_2$ 正则化 | 限制权重增长 |
| **数据增强** | 裁剪/翻转/颜色扰动 | 增加有效训练数据量 |
| **LRN** | 局部神经元竞争 | 增强泛化（后被 BN 替代） |

> Dropout 的直观理解：每次训练随机"关闭"一半神经元，迫使每个神经元都得学会独立解决问题。测试时所有神经元都开启，但输出需要乘以 0.5——相当于对多个子网络做了"模型集成"。

### 详细训练配置

| 参数 | 值 | 说明 |
|------|-----|------|
| 数据集 | ImageNet ILSVRC 2012 | 120 万图 / 1000 类 |
| 输入尺寸 | 224×224×3 | 从 256×256 随机裁剪 |
| 优化器 | SGD + Momentum (0.9) | — |
| 初始学习率 | 0.01 | 手动衰减 |
| 学习率衰减 | 误差不降时 ÷10 | 共衰减 3 次 |
| Batch Size | 128 | — |
| 权重衰减 | 5e-4 | $L_2$ 正则化 |
| Dropout | 0.5 | FC6 和 FC7 层 |
| Epoch | ~90 | 在 ImageNet 上训练约 5-6 天 |
| GPU | 2× GTX 580 (3GB) | 模型并行 |

### 预训练性能

| 模型 | ImageNet Top-1 | ImageNet Top-5 | 参数量 |
|------|---------------|---------------|--------|
| 传统方法 (SIFT+FVs) | — | 26.2% | — |
| **AlexNet** | **~37.5%** (Top-5 错误率 15.3%) | **15.3%** | 60M |
| AlexNet (5 模型集成) | — | **16.4%** | 5×60M |

> 单模型的 15.3% Top-5 错误率比第二名（26.2%）低了将近一半——这在当时是"降维打击"级别的进步。

### 预训练的实用价值

1. **深度学习革命的起点**：证明了监督预训练在大规模数据集上可以学出强大的通用特征
2. **ImageNet 预训练范式的开创者**：随后 10 年，几乎所有视觉模型都从 ImageNet 预训练开始
3. **GPU 深度学习的奠基者**：开创了 GPU 训练深度网络的实践路线
4. **ReLU + Dropout 的经典组合**：被后续数万篇论文采用
5. **迁移学习的第一批实践者**：AlexNet 预训练特征在 PASCAL VOC 等数据集上直接使用，远超传统方法
