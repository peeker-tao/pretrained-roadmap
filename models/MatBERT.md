# MatBERT (Materials BERT)

## 基本信息

- **代表性论文**: 
  - [MatSciBERT: A Materials Domain Language Model for Text Mining and Information Extraction](https://arxiv.org/abs/2109.08178)
  - [MaterialsBERT: A Pretrained Language Model for Materials Science](https://www.nature.com/articles/s41524-022-00719-1)
- **领域**: 材料科学 × NLP

## 创新点

1. **材料科学领域 BERT**: 在材料科学文献上预训练的领域语言模型
2. **材料实体识别**: 识别化学式、材料名称、属性等实体
3. **知识抽取**: 从海量材料科学文献中自动提取结构化知识

## 核心原理

### 预训练数据

- 数百万篇材料科学论文全文
- 包括 ACS、RSC、Elsevier 等出版商的期刊

### 预训练策略

- **MLM**: 材料科学特定的掩码策略（化学式、数字不拆分）
- **化学式感知**: 特殊处理化学式（如 LiFePO₄）作为单一 token

## 预训练方法

### 核心思想：一个材料科学家穷其一生能读完一万篇论文——而材料科学领域有几百万篇论文。MatBERT 通过领域 MLM 预训练"阅读"了数百万篇论文，学会了识别化学式、材料名称、合成条件、物理性质——成为材料科学的"AI 阅读助手"

MatBERT 代表了 NLP 预训练在垂直科学领域的应用范式：通用预训练（SciBERT）→ 领域继续预训练（材料科学文献）→ 领域任务微调（实体识别、关系抽取、合成预测）。

> MatBERT = SciBERT 初始化 + 材料科学文献继续预训练 + 化学式感知 tokenizer + 领域 MLM。输出：材料科学领域的基础语言模型，可用于知识抽取和文献挖掘。

### 训练流水线（Step by Step）

#### Step 1 — 材料科学文献语料的构建

预训练数据是专门从材料科学出版物中构建的：

| 数据源 | 规模 | 时间范围 |
|--------|------|---------|
| ACS Publications | ~500K 论文 | 2000-2020 |
| RSC (Royal Society of Chemistry) | ~300K 论文 | 2000-2020 |
| Elsevier (ScienceDirect) | ~800K 论文 | 2000-2020 |
| Springer Nature | ~400K 论文 | 2000-2020 |
| arXiv (cond-mat) | ~50K 预印本 | 1991-2020 |
| **总计** | **~2M+ 论文** | — |

**数据预处理**：
- 提取全文（非仅摘要）
- 保留化学式、数字、单位和表格
- 材料科学特有的句子分割（化学式中的 `.` 不是句号）

#### Step 2 — 化学式感知的 Tokenizer

材料科学的文本与通用 NLP 不同：

| 挑战 | 通用 Tokenizer 的问题 | MatBERT 的处理 |
|------|---------------------|---------------|
| 化学式 `LiFePO₄` | 被拆成 `[Li, Fe, PO, ₄]` → 失去完整性 | **保留为单一 token** |
| 单位 `2.5 eV` | `[2, ., 5, e, V]` → 无意义的碎片 | **保留为 `2.5_eV`** |
| 材料名 `SrTiO₃` | 被词缀拆分 | **保留为专有 token** |
| 数字 `3.14×10⁻⁸` | 完全破碎 | **保留科学记数法** |

**化学式感知 tokenizer 的构建**：

```text
原始文本: "LiFePO₄ has a band gap of 3.7 eV..."
正则规则:
  LiFePO₄ → [LiFePO₄]（化学式检测 → 单个 token）
  3.7 eV → [3.7_eV]（数值+单位 → 单个 token）

结果: ["[LiFePO₄]", "has", "a", "band", "gap", "of", "[3.7_eV]", "..."]
```

#### Step 3 — 领域继续预训练

MatBERT 不是在材料文献上从头预训练——而是从 SciBERT（在科学文献上预训练的 BERT）出发进行领域继续预训练：

```text
阶段 1: SciBERT 预训练
  数据: 114 万篇科学论文 (Semantic Scholar)
  损失: MLM + NSP
  → 通用的科学语言理解

阶段 2: MatBERT 领域预训练
  初始化: SciBERT 权重
  数据: ~200 万篇材料科学论文
  损失: MLM（动态 masking）
  额外学习: 材料领域术语和语义
  → 材料科学特化的语言模型
```

**领域预训练的必要性**：

| 语句 | SciBERT 理解 | MatBERT 理解 |
|------|------------|------------|
| "The band gap is 2.1 eV" | ✓（通用科学） | ✓✓（理解带隙在材料中的重要性） |
| "Doping with Cr increases Tc" | 部分 | ✓✓（知道 Cr=铬掺杂, Tc=居里温度） |
| "Perovskite ABO₃ structure" | 中等 | ✓✓（知道钙钛矿是材料大类） |

> SciBERT 是"科学通才"，MatBERT 是"材料专才"——两者领域角度不同。

#### Step 4 — 完整训练配置

| 参数 | MatBERT | 说明 |
|------|---------|------|
| 初始化 | SciBERT (scivocab) | 110M 参数 |
| 领域数据 | ~2M 材料论文 | 全文非仅摘要 |
| 模型架构 | BERT-base | 12 层, 768 维 |
| Tokenizer | SciBERT + 化学式规则 | 特殊 token 处理 |
| MLM 掩码率 | 15% | 动态 masking |
| 序列长度 | 512 | — |
| 总训练步数 | 150K | 继续预训练 |
| 优化器 | AdamW | — |
| 学习率 | 5e-5 | 小（微调式继续预训练） |
| Batch Size | 256 | — |

#### Step 5 — 下游材料科学任务

| 任务 | 输入 | 输出 | MatBERT 提升 |
|------|------|------|------------|
| 材料实体识别（NER） | 文本段落 | 化学式、材料名、性质、方法 | +10% vs SciBERT |
| 掺杂信息抽取 | 合成段落 | 基体材料 + 掺杂元素 + 浓度 | +15% |
| 合成路线预测 | 材料名 | 可能的合成方法 | +12% |
| 性质-材料关联 | 性质描述 | 对应的材料列表 | +8% |
| 材料相似度 | 两篇摘要 | 是否讨论相似材料 | +5% |

### MatBERT vs ChemBERTa vs General BERT

| 维度 | BERT-base | SciBERT | ChemBERTa | **MatBERT** |
|------|----------|---------|----------|------------|
| 预训练数据 | 维基+图书 | 科学论文 | **SMILES（分子）** | **材料文献** |
| 数据类型 | 通用文本 | 科学文本 | 化学字符串 | 材料文本 |
| 化学式感知 | ✗ | 中 | ✓✓（SMILES 原生） | **✓✓** |
| 材料领域 | ✗ | 弱 | 分子领域 | **✓✓** |
| 任务范围 | 通用 NLP | 科学 NLP | 分子性质预测 | 材料知识抽取 |

### 预训练的实用价值

1. **材料科学文献的自动化阅读**：数百万论文 → AI 提取关键信息 → 加速材料研究
2. **领域预训练的范式示范**：通用→科学→材料 的三级预训练策略
3. **化学式感知的关键性**：证明了在科学领域，tokenization 的领域定制是必要的
4. **知识图构建**：NER → 关系抽取 → 材料知识图谱 → 数据驱动的材料发现
5. **与 GNoME 互补**：MatBERT 从文本中抽取知识，GNoME 从晶体结构中预测性质
