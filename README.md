## 🧭 方法分类总览（核心分类法）

```
预训练方法
├── 监督预训练
│   ├── 分类目标 (ImageNet Pre-train)
│   ├── 度量学习目标 (FaceNet, ArcFace)
│   └── 弱监督 (Noisy Student, BiT)
│
├── 生成式自监督
│   ├── 自回归 (GPT, LLaMA)
│   ├── 掩码语言建模 (BERT, RoBERTa, T5)
│   ├── 掩码图像建模 (MAE, BEiT, SimMIM)
│   └── 去噪自编码器 (DAE)
│
├── 对比式自监督
│   ├── 实例判别 (SimCLR, MoCo)
│   ├── 聚类对比 (SwAV, DeepCluster)
│   └── 跨模态对比 (CLIP, ALIGN, SigLIP)
│
├── 非对比式
│   ├── 自蒸馏 (BYOL, DINO, DINOv2)
│   └── 协方差正则化 (Barlow Twins, VICReg)
│
└── 多模态
    ├── 视觉-语言对比 (CLIP, SigLIP)
    ├── 视觉-语言生成 (BLIP-2, LLaVA)
    ├── 统一多模态 (GPT-4V, Gemini)
    └── 视频/3D/音频 (VideoMAE, Point-BERT, AudioMAE)
```

## 📊 范式演进

| 阶段 | 时间 | 关键事件 |
|------|------|----------|
| 监督预训练 | 2012–2017 | AlexNet → ResNet, ImageNet 预训练成 CV 标配 |
| 自监督革命 | 2018–2021 | GPT/BERT (NLP) → SimCLR/MoCo/MAE (CV) |
| 基础模型时代 | 2021–至今 | CLIP, GPT-4, Gemini, 多模态统一 |
