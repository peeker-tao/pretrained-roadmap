# MAML (Model-Agnostic Meta-Learning)

## 基本信息

- **论文**: [Model-Agnostic Meta-Learning for Fast Adaptation of Deep Networks](https://arxiv.org/abs/1703.03400)
- **作者**: Chelsea Finn et al. (UC Berkeley)
- **发表**: ICML 2017

## 创新点

1. **模型无关的元学习**: 与任何基于梯度的模型兼容
2. **学习如何学习**: 学习一个初始化，使其在少量梯度步后就能适应新任务
3. **内循环 + 外循环**: 双循环优化框架

## 核心原理

### MAML 算法

**元训练阶段**:
1. 采样一个 batch 的任务
2. 对每个任务：内循环（1-5 步梯度下降）→ 计算任务验证损失
3. 外循环：对所有任务的验证损失求梯度，更新元参数

**元测试阶段**:
1. 在新任务上做少量梯度步
2. 快速适应（通常 1-10 步）

### 数学形式

$$\\theta^* = \\arg\\min_\\theta \\sum_{\\mathcal{T}_i} \\mathcal{L}_{\\mathcal{T}_i}(\\theta - \\alpha \\nabla_\\theta \\mathcal{L}_{\\mathcal{T}_i}^{\\text{train}}(\\theta))$$

## 元训练方法（与预训练的关系）

### 核心思想：预训练教你"知道哪些知识是通用的"——比如学会识别猫的通用特征。MAML 元训练教你"知道如何学习"——给你 5 张"犰狳"的图片，你能在几秒内学会识别犰狳。预训练追求"知识迁移"，MAML 追求"学习能力的迁移"

MAML（Model-Agnostic Meta-Learning）是元学习领域最具影响力的方法。它不预训练一个"强模型"——它预训练一个"强学习者"，即一个模型初始化 $\theta$，从这个初始化出发，仅需少量梯度步就能快速适应新任务。

> MAML = 内循环（任务级快速适应，1-5 步 SGD）+ 外循环（元优化，找到最佳初始化 $\theta^*$）+ 跨任务梯度。预训练找到"好参数"，MAML 找到"好起步点"。

### 训练流水线（Step by Step）

#### Step 1 — 任务分布（Task Distribution）

MAML 训练需要任务分布——从相似但不同的任务中学习"如何学习"：

**Few-Shot 分类中的任务采样**：

```text
元训练集: ImageNet (1000 类)

每轮采样一个 "N-way K-shot" 任务:
  随机选 N 个类（如 N=5）
  每类选 K 个样本作为 Support Set（如 K=1 或 5），
  每类再选 Q 个样本作为 Query Set
```

| 参数 | 含义 | 典型值 |
|------|------|--------|
| N-way | 每任务有 N 个类别 | 5 |
| K-shot | 每类 K 个训练样本 | 1, 5, 10 |
| Support Set | 内循环训练数据 | N×K 个样本 |
| Query Set | 外循环验证数据 | N×Q 个样本 |

#### Step 2 — 内循环：单个任务的快速适应

```text
对于任务 T_i:

输入: Support Set (N×K 个样本)
  1. 从初始参数 θ 出发
  2. 计算 Support Set 上的梯度: ∇_θ L_{T_i}(θ)
  3. 更新: θ'_i = θ - α · ∇_θ L_{T_i}(θ)
  4. 可能重复 1-5 步

输出: 适应后的参数 θ'_i
```

**α 是什么？** 内循环学习率——通常设为 0.01-0.1，比正常训练的学习率大得多（因为只需要走 1-5 步）。

#### Step 3 — 外循环：跨任务的元优化

```text
外循环目标: 找到 θ，使所有任务的适应后性能最好

对于一批任务 {T_1, T_2, ..., T_B}:
  
  对于每个 T_i:
    θ'_i = θ - α · ∇_θ L^{support}_{T_i}(θ)   (内循环)
  
  元损失: L_meta = Σ_i L^{query}_{T_i}(θ'_i)    (用适应后的参数评估)
  
  元更新: θ ← θ - β · ∇_θ L_meta                (外循环)
```

**关键：二阶梯度！**

$$\nabla_\theta \mathcal{L}_{\text{meta}} = \nabla_\theta \sum_i \mathcal{L}_{T_i}(\theta - \alpha \nabla_\theta \mathcal{L}_{T_i}(\theta))$$

这涉及到**梯度的梯度**（二阶导数）——MAML 需要对内循环的整个优化路径求导。这计算量很大，实践中常用一阶近似（FOMAML，忽略二阶项）。

#### Step 4 — MAML vs 标准预训练（关键区别）

| 维度 | 标准预训练 | MAML |
|------|----------|------|
| 优化目标 | $\min_\theta \mathbb{E}_{x,y} [\mathcal{L}(f_\theta(x), y)]$ | $\min_\theta \mathbb{E}_{\mathcal{T}} [\mathcal{L}(\text{Adapt}(\theta), \mathcal{T})]$ |
| 评价标准 | 在**当前**任务上表现好 | **适应后**在新任务上表现好 |
| 训练方式 | 大量数据上的监督/自监督 | 大量**任务**上的元学习 |
| 适应阶段 | Fine-tune（数百-数千步） | **快速适应（1-10 步 SGD）** |
| 数据需求 | 每个任务大量数据 | 每个任务仅少量数据 |

> 标准预训练学习"好参数"，MAML 学习"好起点"——从这个起点一步就能到达任何新任务的"好参数"。

#### Step 5 — 完整元训练流程

```text
初始化: θ
循环 (外循环):
  采样一批任务 {T_1, ..., T_B}
  
  对于每个 T_i:
    # 内循环
    θ'_i = θ
    for step in range(5):
      θ'_i = θ'_i - α · ∇ Loss(support_i, θ'_i)
    
    # 计算查询损失
    loss_i = Loss(query_i, θ'_i)
  
  # 外循环
  θ = θ - β · ∇_θ Σ loss_i
  
收敛后:
  θ* = 最佳初始化（"最佳学习者"的起点）
```

### 元测试阶段

```text
新任务（从未见过的类别，如"犰狳"vs"犰狳近亲"）:

Support Set: 5 张犰狳图片 + 5 张犰狳近亲图片
  → 从 θ* 出发，1-5 步 SGD
  → 得到 适应后的参数 θ'（专为这个 5-shot 任务优化）

Query Set: 测试"学会了没有"
```

### MAML 的变体

| 变体 | 方法 | 优势 |
|------|------|------|
| FOMAML | 忽略二阶梯度 | 更快，效果接近 |
| Reptile | 简单的均值更新 | 更简单 |
| Meta-SGD | 学习每组参数的学习率 | 更灵活 |
| MAML++ | 改进训练稳定性 | 更稳定 |

### 详细训练配置

| 参数 | MAML (Mini-ImageNet) | 说明 |
|------|-------------------|------|
| 任务分布 | Mini-ImageNet (64 类训练, 16 类验证, 20 类测试) | 来自 ImageNet |
| N-way K-shot | 5-way 1-shot / 5-way 5-shot | — |
| 内循环步数 | 5 | 适应步数 |
| 内循环学习率 α | 0.01 | 较大（快速适应） |
| 外循环学习率 β | 0.001 | 标准 |
| 外循环任务数 | 4 | batch of tasks |
| 总迭代 | 60K | — |
| 架构 | 4 层 CNN | 模型无关 |

### 元训练的实用价值

1. **少样本学习的里程碑**：10 张图片就能学会新类别 → 类似人类的"举一反三"
2. **"学会学习"思想的推广**：MAML 启发了元强化学习、元模仿学习
3. **与预训练的互补**：预训练提供好的"知识起点"，MAML 提供好的"学习起点" → 可以结合
4. **快速适应**：1-5 步梯度就能达到较好精度 → 适用于需要快速定制化的场景
5. **元学习的思想遗产**：虽然 MAML 本身不直接用于大规模预训练，但"优化优化过程"的思想深刻影响了后续的 LoRA、Adapter 等方法
