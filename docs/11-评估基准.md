# 第 11 章：评估基准与数据集

评估是推动预训练方法进步的关键。不同的评估方法反映表征质量的不同维度。

---

## 11.1 视觉评估

### 线性探测 (Linear Probing)

冻结预训练编码器的权重，在 ImageNet 或其他分类数据集上训练线性分类器。

| 方法 | ImageNet Top-1 | 年份 |
|------|---------------|------|
| 监督 ResNet-50 | 76.5% | 2016 |
| SimCLR (ResNet-50) | 69.3% | 2020 |
| MoCo v2 (ResNet-50) | 71.1% | 2020 |
| BYOL (ResNet-50) | 74.3% | 2020 |
| SwAV (ResNet-50) | 75.3% | 2020 |
| DINO (ViT-B/8) | 78.2% | 2021 |
| MAE (ViT-L/16) | 76.0% | 2021 |
| DINOv2 (ViT-g/14) | **83.5%** | 2023 |

### 半监督评估

仅使用 1% 或 10% 的 ImageNet 标签微调。**反映预训练表征的数据效率**。

| 方法 | 1% ImageNet | 10% ImageNet |
|------|------------|-------------|
| Supervised ResNet-50 | 25.4% | 56.4% |
| SimCLR | 48.3% | 65.6% |
| BYOL | 53.2% | 68.8% |
| SwAV | — | 72.7% |

### 迁移学习评估

**VTAB (Visual Task Adaptation Benchmark)** — [[论文]](https://arxiv.org/abs/1910.04867)

VTAB 包含 19 个数据集，分为 3 组：

| 分组 | 描述 | 代表数据集 |
|------|------|-----------|
| **Natural** | 自然图像 | CIFAR-100, Caltech-101, Oxford Pets, Flowers |
| **Specialized** | 专业图像 | Medical, Satellite, Textures |
| **Structured** | 结构化理解 | dSprites, CLEVR, ShapeNet |

### 检测/分割评估

| 基准 | 任务 | 评估指标 |
|------|------|---------|
| **COCO** | 目标检测、实例分割 | AP, AP50, AP75 |
| **ADE20K** | 语义分割 | mIoU |
| **Cityscapes** | 街景语义分割 | mIoU |
| **LVIS** | 长尾实例分割 | AP |

### 关键发现

1. **线性探测 ≠ 全量微调**：有些方法线性探测好但微调差（对比学习），反之亦然（MIM）
2. **没有单一指标能全面衡量表征质量**
3. **大规模预训练在所有指标上都更好**

## 11.2 NLP 评估

### 语言理解

#### GLUE (General Language Understanding Evaluation)

- **论文**：[GLUE: A Multi-Task Benchmark and Analysis Platform for Natural Language Understanding](https://arxiv.org/abs/1804.07461)
- 9 项任务：情感分析、文本蕴含、问题回答等
- 评估指标：各任务准确率/匹配度

| 方法 | GLUE 得分 |
|------|----------|
| BERT-Base | 79.6 |
| BERT-Large | 82.1 |
| RoBERTa | 87.6 |
| XLNet | 89.0 |
| ALBERT | 89.4 |

#### SuperGLUE

- **论文**：[SuperGLUE: A Stickier Benchmark for General-Purpose Language Understanding Systems](https://arxiv.org/abs/1905.00537)
- GLUE 的升级版，更困难的任务

### 推理评估

#### MMLU (Massive Multitask Language Understanding)

- **论文**：[Measuring Massive Multitask Language Understanding](https://arxiv.org/abs/2009.03300)
- 57 个学科，约 14000 个问题
- 评估模型的广度和深度知识

| 模型 | MMLU 得分 |
|------|----------|
| GPT-3 (175B) | 43.9% |
| LLaMA-65B | 63.4% |
| GPT-4 | 86.4% |
| Gemini Ultra | 90.0% |
| LLaMA-3-405B | 89.0% |

#### BIG-bench

- **论文**：[Beyond the Imitation Game: Quantifying and extrapolating the capabilities of language models](https://arxiv.org/abs/2206.04615)
- 204 项任务，评估语言模型的多方位能力
- 首次系统性研究"涌现能力"

#### GSM8K (数学推理)

- **论文**：[Training Verifiers to Solve Math Word Problems](https://arxiv.org/abs/2110.14168)
- 小学水平的数学应用题
- 测试模型的数学推理能力

### 文本生成

| 指标 | 描述 | 局限 |
|------|------|------|
| **BLEU** | 基于 n-gram 精确率 | 忽略语义相似性 |
| **ROUGE** | 基于 n-gram 召回率 | 忽略语法流畅性 |
| **BERTScore** | 基于 BERT 嵌入的语义相似度 | 计算开销大 |
| **Perplexity** | 模型对下一个词的困惑度 | 只反映语言建模质量 |

## 11.3 多模态评估

### 零样本分类 (Zero-shot Image Classification)

| 方法 | ImageNet 零样本 |
|------|----------------|
| CLIP (ResNet-50) | 58.2% |
| CLIP (ViT-L) | 76.2% |

### 图文检索 (Image-Text Retrieval)

**Recall@K**：在前 K 个结果中命中的概率。

| 方法 | MSCOCO R@1 (I→T) | MSCOCO R@1 (T→I) |
|------|-----------------|-----------------|
| CLIP (ViT-L) | 58.4% | 37.8% |
| ALIGN | 58.6% | 45.6% |
| BLIP | 65.1% | 52.3% |

| 方法 | VQAv2 test-std | 说明 |
|------|---------------|------|
| BLIP-2 | 82.0% | Q-Former + 冻结 LLM |
| LLaVA-1.5 | 80.0% | 简单投影层 |

### 多模态推理

| 基准 | 描述 | 典型指标 |
|------|------|---------|
| **MMBench** | 20 个细粒度能力维度 | 准确率 |
| **MMMU** | 大学级多模态理解 | 准确率 |

## 11.4 评估方法对比

| 评估类型 | 所测维度 | 适合场景 |
|---------|---------|---------|
| 线性探测 | 表征的线性可分性 | 快速验证、比较方法 |
| 全量微调 | 表征的表达能力 | 实际应用评估 |
| 半监督 | 数据效率 | 低资源场景 |
| 零样本 | 泛化能力 | 基础模型评估 |
| 少样本 | 适应效率 | 大模型评估 |
| 迁移学习 | 泛化广度 | 综合评价 |
