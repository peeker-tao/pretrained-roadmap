# AlphaFold

## 基本信息

- **论文**: [Highly Accurate Protein Structure Prediction with AlphaFold](https://www.nature.com/articles/s41586-021-03819-2)
- **作者**: John Jumper et al. (DeepMind)
- **发表**: Nature, 2021

## 创新点

1. **端到端蛋白质结构预测**: 从氨基酸序列直接预测 3D 结构，准确率达到原子级别
2. **Evoformer**: 结合 MSA（多序列比对）和 Pair 表示的专用 Transformer
3. **IPA (Invariant Point Attention)**: 3D 空间中的 SE(3) 不变注意力机制
4. **CASP14 突破**: 首次达到实验精度级别的蛋白质结构预测

## 核心原理

### 架构

1. **MSA 表示**: 从多序列比对中提取进化信息
2. **Pair 表示**: 残基对之间的距离和方向关系
3. **Evoformer**: MSA 和 Pair 表示的交互更新
4. **结构模块**: IPA 预测 3D 坐标

### Evoformer

- 通过三角更新传播 Pair 信息的几何约束
- MSA 行/列注意力捕获序列协变模式

## 预训练方法

### 核心思想：蛋白质的 3D 结构隐藏在两个信息源中——进化历史（多条相关蛋白质的序列比对）和物理化学约束（残基对的距离/角度关系）。AlphaFold 的 Evoformer 将两者优雅地交织在一起

AlphaFold 的"预训练"与 CV/NLP 中的概念不同——它不是在无标签数据上自监督学习，而是**对进化数据和结构数据进行联合建模**。其训练涉及多个精心设计的数据源和多种损失函数。

> AlphaFold 训练 = PDB 结构数据（~170K）+ MSA 搜索（进化信息）+ 自蒸馏（用自身预测生成更多训练数据）+ 多种损失（FAPE + 角度 + 距离 + MSA 一致性）。核心瓶颈不是模型大小，而是 MSA 搜索的计算开销。

### 训练流水线（Step by Step）

#### Step 1 — 多序列比对（MSA）的构建

AlphaFold 的第一个关键输入是 MSA——在大型蛋白质数据库中搜索进化相关的序列：

```text
目标氨基酸序列: MKFLILFNILVSTLA...

遗传数据库搜索:
  UniRef90 (~150M 序列)  →  快速初筛
  BFD (~2.5B 序列)      →  深度搜索
  MGnify (~1B 序列)      →  宏基因组补充
  
输出: MSA 矩阵 [N_seq × L]
  每行是一个相关蛋白质的序列（对齐到目标序列）
  每列是目标序列的一个位置上的进化变异模式
```

| 数据库 | 规模 | 搜索目的 |
|--------|------|---------|
| UniRef90 | 1.5 亿 | 去除冗余的蛋白质序列 |
| BFD | 25 亿 | 大规模蛋白质序列家族 |
| MGnify | 10 亿 | 宏基因组（环境样本）序列 |
| UniClust30 | 数亿 | 结构模板搜索 |

**MSA 的信息论本质**：同一列中的氨基酸分布反映了该位置的进化约束——如果第 47 位总是疏水氨基酸，那这个位置很可能在蛋白质内部。

#### Step 2 — Evoformer：MSA + Pair 的双流信息处理

Evoformer 是 AlphaFold 的核心组件，处理两个表示：

| 表示 | 形状 | 含义 |
|------|------|------|
| **MSA 表示** | $N_{\text{seq}} \times L \times d_{\text{msa}}$ | 进化信息——哪些位点协同变异 |
| **Pair 表示** | $L \times L \times d_{\text{pair}}$ | 空间关系——残基对之间的距离和方向 |

**信息交换机制**：

```text
MSA 表示 ←→ Pair 表示（通过 Evoformer 的 48 个 Block）

每个 Block 的操作:
1. MSA 行注意力: 不同序列之间交换信息
2. MSA 列注意力: 不同位置之间交换信息（捕获共变模式）
3. 外积更新: MSA 信息注入 Pair（"第 i 位和第 j 位的关系"）
4. 三角更新: 利用 Pair 的几何约束（三角形不等式）
5. 三角自注意力: 通过 Pair 的距离信息增强 MSA
```

**三角更新/三角自注意力**（AlphaFold 的独特设计）：

生物分子中距离满足三角不等式：如果残基 i 靠近 j，j 靠近 k，那么 i 和 k 也不能太远。通过 Pair 矩阵的三角操作来强制这种约束：

$$\text{如果 } d_{ij} \approx 5Å, d_{jk} \approx 5Å \implies d_{ik} \leq 10Å$$

> 类比：Evoformer 就像一个"八卦系统"——MSA 流说"第 20 位和第 45 位总是同时突变（可能空间上靠近）"→ Pair 流接收这个信息，加入"第 20 位和第 45 位的距离应该很近"的约束 → 结构预测更准确。

#### Step 3 — 结构模块（Structure Module）+ IPA

结构模块将 Pair 表示 + 单残基表示转化为 3D 坐标：

**IPA（Invariant Point Attention）** 的关键：标准 Transformer 的注意力对 3D 旋转不敏感——但蛋白质结构是 SE(3) 不变的（任意旋转/平移都是同一个结构）。IPA 设计了专门的 3D 注意力机制来保持这种完备性：

$$\text{IPA}(Q, K, V) = \sum_j \text{softmax}\left(\frac{QK^T}{\sqrt{d}} + \text{Bias}(d_{ij})\right) V_j \cdot T_j$$

其中 $T_j$ 是每个残基的局部坐标系（刚体变换），$d_{ij}$ 是当前预测的残基间距离。

**结构模块的输出流程**：

```text
1. IPA → 更新残基级别的特征
2. 预测每个残基的刚体变换（旋转 + 平移）
3. 应用刚体变换 → 更新所有残基的 3D 坐标
4. 循环 8 次（逐步优化结构）
```

#### Step 4 — 多种训练损失

AlphaFold 使用多任务损失进行端到端训练：

| 损失 | 公式/描述 | 权重 |
|------|---------|------|
| **FAPE**（Frame Aligned Point Error） | 在局部坐标系下计算原子坐标误差 | 主导 |
| 距离直方图 | 预测的残基对距离分布 vs 真实结构 | 0.3 |
| 角度预测 | 预测的键角/二面角 vs 真实结构 | 0.01 |
| MSA 掩码预测 | 类似 BERT 的 MSA 层面 MLM | 2.0 |
| 结构违规 | 键长、键角、范德华的物理约束 | 0.01 |
| pLDDT 置信度 | 预测每个残基的可靠性（自评估） | 0.01 |

**FAPE 损失**（AlphaFold 的核心贡献之一）：

$$L_{\text{FAPE}} = \frac{1}{N} \sum_{i,j} \|T_i^{-1} \circ x_j^{\text{pred}} - T_i^{-1} \circ x_j^{\text{true}}\|$$

其中 $T_i^{-1}$ 将全局坐标转换到残基 i 的局部坐标系——**使损失对全局旋转平移不变**。这是 AlphaFold 能达到原子级精度的关键。

#### Step 5 — 自蒸馏训练

AlphaFold 使用了类似 NoisyStudent 的自蒸馏策略：

```text
Round 1: PDB 真实结构训练 (170K)
  → AlphaFold 预测 Uniclust30 中 350K 序列的结构
  → 筛选置信度高的预测 (pLDDT > 70)

Round 2: PDB (170K) + 高质量预测 (350K) 联合训练
  → 性能提升 ~2-3%
```

**为什么自蒸馏有效？** PDB 中的结构偏向易于结晶的蛋白质（稳定、小尺寸）。自蒸馏扩大了训练数据的多样性，覆盖了更多"困难"蛋白质。

### AlphaFold 的训练 vs 传统预训练

| 维度 | CV/NLP 预训练 | AlphaFold 训练 |
|------|------------|-------------|
| 数据 | 无标签数据（图像/文本） | 有标签（PDB 结构）+ 进化数据（MSA） |
| 目标 | 自监督（MLM/对比） | 多任务监督 + 自蒸馏 |
| 计算瓶颈 | 模型前向传播 | **MSA 搜索（CPU 密集）** |
| 训练时间 | 数天-数周 | ~1 周（128 TPUv3） |
| 最大模型 | AlphaFold 2.2B 参数 | — |

### 详细训练配置

| 参数 | AlphaFold 配置 | 说明 |
|------|--------------|------|
| PDB 训练集 | ~170K 结构 | 实验确定的蛋白质结构 |
| 自蒸馏数据 | ~350K 预测 | Uniclust30 上的高置信度预测 |
| Evoformer Block | 48 | 核心 MSA-Pair 交互模块 |
| MSA 通道数 | 256 | — |
| Pair 通道数 | 128 | — |
| 结构模块循环 | 8 次 | 逐步优化 |
| 优化器 | Adam | — |
| 学习率 | 1e-3 → 1e-4 | — |
| 硬件 | 128 TPUv3 | 训练 ~7 天 |
| 推理时间（含 MSA） | ~10-30 分钟/蛋白质 | MSA 搜索是主要瓶颈 |

### 预训练的实用价值

1. **50 年蛋白质折叠问题的解决**：CASP14 上 GDT > 90%（达到实验精度）
2. **Evoformer 的建筑学贡献**：MSA + Pair 的双流融合设计 → 为多模态信息融合提供了模板
3. **FAPE 损失的设计**：对 SE(3) 变换不变的损失函数 → 适用于所有 3D 分子/材料建模
4. **自蒸馏的应用**：展示了"用模型预测扩展训练数据"在科学计算中的可行性
5. **科学 AI 的范式变革**：从"AI 辅助科学研究"到"AI 主导科学发现"的里程碑
