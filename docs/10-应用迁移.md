# 第 10 章：预训练模型的应用与迁移

预训练的价值最终体现在其**迁移到下游任务的能力**上。本章总结预训练模型适配下游任务的主要方法和迁移特性。

---

## 10.1 下游任务适配方法

### 方法概览

| 方法 | 描述 | 更新参数 | 适用场景 |
|------|------|---------|---------|
| **全量微调 (Full Fine-tuning)** | 在目标任务上更新所有参数 | 全部 | 下游数据充足 |
| **线性探测 (Linear Probing)** | 冻结编码器，仅训练分类头 | 分类头 | 快速评估表征质量 |
| **部分微调** | 只微调最后几层 | 部分 | 中等规模数据 |
| **Adapter** | 在层间插入小网络 | Adapter + 头 | 参数高效，多任务 |
| **Prefix Tuning** | 学习连续提示向量 | 提示向量 | 语言模型适配 |
| **Prompt Tuning** | 学习软提示嵌入 | 提示嵌入 | 冻结 LLM |
| **LoRA** | 低秩适配矩阵 | LoRA 矩阵 | 大模型微调主流 |
| **Visual Prompt** | 输入空间改适 | 输入 / 部分 | 免参数修改 |

### 各方法详述

#### 全量微调 (Full Fine-tuning)

- 最直接的方法，在所有参数上执行反向传播
- 当下游数据与预训练数据分布一致时效果最好
- 缺点：每任务独立保存全部参数，存储成本高

#### 线性探测 (Linear Probing)

- **标准实践**：冻结编码器，训练一个线性分类层
- **作用**：评估预训练表征质量的"金标准"
- **局限**：无法充分利用预训练模型的表达能力

#### LoRA (Low-Rank Adaptation, 2021)

- **论文**：[LoRA: Low-Rank Adaptation of Large Language Models](https://arxiv.org/abs/2106.09685)
- **作者**：Edward Hu et al. (Microsoft)
- **发表**：ICLR 2022
- **核心贡献**：
  - **低秩分解**：$\\Delta W = BA$，其中 $B \\in \\mathbb{R}^{d \\times r}$，$A \\in \\mathbb{R}^{r \\times k}$，$r \\ll \\min(d, k)$
  - 只训练低秩矩阵，冻结原权重
  - 参数量仅为全量微调的 0.1%–1%
- **QLoRA** (Dettmers et al., 2023) — [[论文]](https://arxiv.org/abs/2305.14314)
  - 4-bit 量化 + LoRA
  - 可在消费级 GPU 上微调 65B 模型

#### Adapter (2019)

- **论文**：[Parameter-Efficient Transfer Learning for NLP](https://arxiv.org/abs/1902.00751)
- **作者**：Neil Houlsby et al. (Google)
- **核心贡献**：
  - 在 Transformer 每层插入瓶颈结构（下投影 → 非线性 → 上投影）
  - 冻结原模型，只训练 Adapter
  - 适合多任务场景（每任务仅保存 Adapter 参数）

#### Prefix Tuning (2021)

- **论文**：[Prefix-Tuning: Optimizing Continuous Prompts for Generation](https://arxiv.org/abs/2101.00190)
- **作者**：Xiang Lisa Li, Percy Liang (Stanford)
- **发表**：ACL 2021
- **核心贡献**：
  - 在 Transformer 每层前拼接可学习的"前缀"向量
  - 保持原模型冻结
  - 在生成任务上接近全量微调性能

#### Prompt Tuning (2022)

- **论文**：[The Power of Scale for Parameter-Efficient Prompt Tuning](https://arxiv.org/abs/2104.08691)
- **作者**：Brian Lester et al. (Google)
- **发表**：EMNLP 2021
- **核心贡献**：
  - 仅在输入嵌入层添加可学习的软提示
  - 比 Prefix Tuning 更简单，随模型规模扩大性能提升

### 参数效率对比

| 方法 | 额外参数比例 | 存储多个任务 | 性能接近全量微调 |
|------|------------|------------|----------------|
| 全量微调 | 100% | ❌ | — |
| Adapter | 3–6% | ✅ | ✅（大模型） |
| Prefix Tuning | 0.1–1% | ✅ | ✅（大模型） |
| LoRA | 0.1–1% | ✅ | ✅ |
| Prompt Tuning | 0.01% | ✅ | ✅（超大模型） |

## 10.2 领域迁移分析

### 迁移类型

| 迁移类型 | 描述 | 示例 |
|---------|------|------|
| **同域迁移** | 预训练与下游数据分布一致 | ImageNet → CIFAR-10 |
| **跨域迁移** | 预训练与下游数据分布不同 | ImageNet → 医学图像 |
| **跨任务迁移** | 预训练任务与下游任务不同 | 分类 → 检测 → 分割 |
| **跨语言迁移** | 多语言间的迁移 | 英语预训练 → 中文任务 |

### 影响迁移效果的因素

1. **预训练数据分布**：与下游数据越接近，迁移效果越好
2. **预训练目标函数**：不同目标函数产生不同特性的表征
   - 对比方法 → 线性探测更优
   - 生成方法 → 全量微调更优
3. **模型规模**：大模型通常迁移更好（Scaling Law）
4. **下游数据规模**：下游数据越多，微调收益越大
5. **任务差距**：预训练与下游任务差距越大，需要的资源越多

### 迁移效率

**多少下游数据足够？** 这取决于预训练质量和任务难度：

| 场景 | 所需数据量 | 建议方法 |
|------|-----------|---------|
| 与预训练高度相似 | 几十–几百张 | 线性探测 |
| 中等迁移难度 | 几千张 | 部分微调 |
| 域偏移较大 | 几万张 | 全量微调 |
| 全新领域 | 尽可能多 | 全量微调 + 数据增强 |

## 10.3 零样本与少样本学习

### 零样本学习

**直接应用预训练模型到未见过的任务，无需任何训练样本。**

#### CLIP 的零样本分类

- 将类别名称嵌入到提示模板中（如 "A photo of a {class}"）
- 计算图像编码与所有类别文本编码的相似度
- 在 ImageNet 上达到 76.2% 的零样本准确率（ViT-L）

#### GPT 的零样本推理

- 通过自然语言指令描述任务
- 无需示例模型即可执行

### 少样本学习

**仅给模型少量（1–100 个）标注样本即学习新任务。**

#### In-Context Learning (GPT-3)

- **论文**：[Language Models are Few-Shot Learners](https://arxiv.org/abs/2005.14165)
- 在输入中提供少量示例（输入 → 输出格式）
- 模型通过模式匹配"学会"任务，无需更新参数
- 随模型规模增加性能显著提升

#### MAML (Model-Agnostic Meta-Learning, 2017)

- **论文**：[Model-Agnostic Meta-Learning for Fast Adaptation of Deep Networks](https://arxiv.org/abs/1703.03400)
- **作者**：Chelsea Finn, Pieter Abbeel, Sergey Levine (UC Berkeley)
- **发表**：ICML 2017
- **核心贡献**：
  - 学习一个"容易微调"的初始化
  - 通过元学习（Meta-Learning）实现快速适应

### 涌现能力

涌现能力（Emergent Abilities）是指**在小模型中不存在、但在大模型中突然出现的能力**：

- **In-Context Learning**：在约 10²² FLOPs 时涌现
- **思维链 (Chain-of-Thought)**：在约 10²³ FLOPs 时涌现
- **指令跟随**：在约 10²³ FLOPs 时涌现

这些能力是预训练模型规模扩展带来的最令人兴奋的现象。

## 10.4 本章小结

| 适配方法 | 适用条件 | 成本 | 效果 |
|---------|---------|------|------|
| 线性探测 | 数据极少 | 极低 | 基线性能 |
| 全量微调 | 数据充足 | 高 | 最高 |
| LoRA | 大模型 + 一般数据 | 低 | 接近全量微调 |
| In-Context Learning | 超大模型 | 无 | 有上限 |
| Prompt Tuning | 超大模型 + 多任务 | 极低 | 规模越大越好 |
