# 第 6 章：对比式自监督预训练

对比式自监督学习的核心思想是：**在表示空间中拉近相似样本（正例对）、推远不相似样本（负例对）**。这是 CV 领域自监督学习最重要的范式之一。

---

## 6.1 实例判别 (Instance Discrimination)

### 核心思想

**将每个样本视为一个独立的类别**。对于一张图片，不同的数据增强视图被视为正例对，而不同图片的视图被视为负例对。

损失函数（InfoNCE）：

$$
\mathcal{L} = -\log \frac{\exp(\operatorname{sim}(z_i, z_j)/\tau)}{\sum_{k=1}^{2N} \mathbb{1}_{[k \neq i]} \exp(\operatorname{sim}(z_i, z_k)/\tau)}
$$

其中 $\operatorname{sim}(\cdot, \cdot)$ 为余弦相似度，$\tau$ 为温度系数。

### 经典模型

#### [SimCLR](../models/SimCLR.md) (2020)

- **论文**：[A Simple Framework for Contrastive Learning of Visual Representations](https://arxiv.org/abs/2002.05709)
- **作者**：Ting Chen, Simon Kornblith, Mohammad Norouzi, Geoffrey Hinton (Google)
- **发表**：ICML 2020
- **核心贡献**：
  - **简洁而完整**的框架：数据增强 → 编码器 → 投影头 → InfoNCE Loss
  - **系统性消融研究**揭示了关键组件：
    - 数据增强的**决定性作用**（随机裁剪 + 色彩抖动最关键）
    - 投影头（Projection Head）的重要性
    - 大 batch size (4096+) 的必要性
  - **非线性投影头**显著提升表征质量

#### [SimCLR v2](../models/SimCLRv2.md) (2020)

- **论文**：[Big Self-Supervised Models are Strong Semi-Supervised Learners](https://arxiv.org/abs/2006.10029)
- **作者**：Ting Chen et al. (Google)
- **发表**：NeurIPS 2020
- **核心贡献**：
  - 更大的 [ResNet](../models/ResNet.md) 模型（152 层 + SK 分支）
  - 更深的投影头（3 层 MLP）
  - 自监督预训练 → 少样本微调 → 自蒸馏的三阶段框架

#### [MoCo](../models/MoCo.md) (Momentum Contrast, 2020)

- **论文**：[Momentum Contrast for Unsupervised Visual Representation Learning](https://arxiv.org/abs/1911.05722)
- **作者**：Kaiming He, Haoqi Fan, Yuxin Wu, Saining Xie, Ross Girshick (Meta)
- **发表**：CVPR 2020
- **核心贡献**：
  - **动量编码器**（Momentum Encoder）：对编码器参数做指数滑动平均
  - **队列机制**（Queue）：存储大量负例表征，复用计算结果
  - **解耦负例数量与 batch size**：可用小 batch 训练

#### [MoCo v2](../models/MoCo.md) (2020)

- **论文**：[Improved Baselines with Momentum Contrastive Learning](https://arxiv.org/abs/2003.04297)
- **作者**：Xinlei Chen, Haoqi Fan, Ross Girshick, Kaiming He (Meta)
- **发表**：arXiv, 2020
- **核心贡献**：
  - 吸收 [SimCLR](../models/SimCLR.md) 的设计：MLP 投影头 + 更强的数据增强
  - 不增加计算量即可显著提升性能
  - 证明了各组件在不同框架中的通用性

#### [MoCo v3](../models/MoCo.md) (2021)

- **论文**：[An Empirical Study of Training Self-Supervised Vision Transformers](https://arxiv.org/abs/2104.02057)
- **作者**：Xinlei Chen, Saining Xie, Kaiming He (Meta)
- **发表**：ICCV 2021
- **核心贡献**：
  - 将 [MoCo](../models/MoCo.md) 扩展到 [ViT](../models/ViT.md) 架构
  - 发现 [ViT](../models/ViT.md) 在自监督训练中的不稳定问题
  - 通过**冻结 patch 投影层**稳定训练

#### [NPID](../models/NPID.md) (2018)

- **论文**：[Unsupervised Feature Learning via Non-Parametric Instance-level Discrimination](https://arxiv.org/abs/1805.01978)
- **作者**：Zhirong Wu et al. (CMU / Meta)
- **发表**：CVPR 2018
- **核心贡献**：
  - 首次明确提出"实例判别"范式
  - 非参数 Softmax + 内存缓存（Memory Bank）
  - 对比学习的早期奠基工作

#### [CPC](../models/CPC.md) / Contrastive Predictive Coding (2018)

- **论文**：[Representation Learning with Contrastive Predictive Coding](https://arxiv.org/abs/1807.03748)
- **作者**：Aaron van den Oord, Yazhe Li, Oriol Vinyals (DeepMind)
- **发表**：arXiv, 2018
- **核心贡献**：
  - 提出 **InfoNCE Loss**（对比学习的基础损失函数）
  - 预测未来潜表征而非原始信号
  - 跨模态（图像、语音、文本）的通用框架

#### [AMDIM](../models/AMDIM.md) (2019)

- **论文**：[Learning Representations by Maximizing Mutual Information Across Views](https://arxiv.org/abs/1906.00910)
- **作者**：Philip Bachman, R. Devon Hjelm, William Buchwalter
- **发表**：NeurIPS 2019
- **核心贡献**：
  - 最大化不同特征尺度间的互信息
  - 多头（Multi-Head）预测

### 关键设计选择

| 设计维度 | 选项 | 影响 |
|---------|------|------|
| **正负例定义** | 同一图片的增强 vs. 不同图片 | 决定学习目标 |
| **数据增强** | 裁剪、色彩抖动、高斯模糊、旋转 | 最关键的因素 |
| **投影头** | 线性 / 1层 / 2层 / 3层 MLP | 影响线性探测性能 |
| **负例数量** | 小 (256) ~ 大 (65536) | 更多通常更好 |
| **损失函数** | InfoNCE / NT-Xent / NCE | 影响梯度特性 |
| **温度系数** | $\\tau \\in [0.1, 1.0]$ | 控制聚类的密度 |

## 6.2 聚类对比 (Clustering-based Contrastive)

### 核心思想

**在对比学习框架中引入聚类，将属于同一簇的样本视为正例，利用数据分布的结构信息作为隐式监督。**

### 经典模型

#### [DeepCluster](../models/DeepCluster.md) (2018)

- **论文**：[Deep Clustering for Unsupervised Learning of Visual Features](https://arxiv.org/abs/1807.05520)
- **作者**：Mathilde Caron et al. (INRIA / Facebook)
- **发表**：ECCV 2018
- **核心贡献**：
  - 交替执行：特征聚类（k-means）+ 学习分类器
  - 简单有效，但需要小心处理**崩溃问题**
  - 启发后续聚类对比工作

#### [SwAV](../models/SwAV.md) (2020)

- **论文**：[Unsupervised Learning of Visual Features by Contrasting Cluster Assignments](https://arxiv.org/abs/2006.09882)
- **作者**：Mathilde Caron et al. (INRIA / Facebook)
- **发表**：NeurIPS 2020
- **核心贡献**：
  - **在线聚类**：Sinkhorn-Knopp 算法平衡聚类分配
  - **交换预测**：一个视图的编码经聚类后，预测另一个视图的聚类分配
  - **多分辨率裁剪**（Multi-Crop）：用小分辨率裁剪增加样本多样性
  - 不需要 Memory Bank 或大 batch

#### [SeLa](../models/SeLa.md) (2020)

- **论文**：[Self-labelling via Simultaneous Clustering and Representation Learning](https://arxiv.org/abs/1911.05371)
- **作者**：Yuki M. Asano, Christian Rupprecht, Andrea Vedaldi (Oxford)
- **发表**：ICLR 2020
- **核心贡献**：
  - 自标注（Self-Labelling）：将聚类标签作为伪标签
  - 约束聚类分配为均匀分布
  - 交替优化

### 聚类对比的优势与局限

| 优势 | 局限 |
|------|------|
| 利用数据分布的结构信息 | 聚类质量不稳定 |
| 不需要显式负例 | 容易陷入平凡解（崩溃） |
| 可扩展到大规模数据 | 超参数敏感 |
| 产生语义化的聚类结果 | 对初始化敏感 |

## 6.3 跨模态对比学习

### 核心思想

**利用另一个模态的语义作为监督信号。** 最成功的例子是利用文本作为视觉信号的监督——对应图文对的对比预训练。

### 经典模型

#### [CLIP](../models/CLIP.md) (2021)

- **论文**：[Learning Transferable Visual Models From Natural Language Supervision](https://arxiv.org/abs/2103.00020)
- **作者**：Alec Radford et al. (OpenAI)
- **发表**：ICML 2021
- **核心贡献**：
  - **4 亿图文对**的对比预训练
  - **双塔架构**：独立的图像编码器和文本编码器
  - **零样本图像分类**能力：通过"图片 vs. 类别文本"的相似度匹配
  - 在 30+ 个数据集上的零样本迁移能力
  - 无需微调即可达到 [ResNet](../models/ResNet.md)-50 监督性能

#### [ALIGN](../models/ALIGN.md) (2021)

- **论文**：[Scaling Up Visual and Vision-Language Representation Learning With Noisy Text Supervision](https://arxiv.org/abs/2102.05918)
- **作者**：Chao Jia et al. (Google)
- **发表**：ICML 2021
- **核心贡献**：
  - **10 亿噪声图文对**（无需人工清洗）
  - 验证了**规模可以补偿噪声**的假设
  - 大规模双塔对比框架

#### [SigLIP](../models/SigLIP.md) (2023)

- **论文**：[Sigmoid Loss for Language Image Pre-Training](https://arxiv.org/abs/2303.15343)
- **作者**：Xiaohua Zhai et al. (Google)
- **发表**：ICCV 2023
- **核心贡献**：
  - **Sigmoid Loss** 替代 Softmax + InfoNCE
  - 不再需要全局归一化，每对独立计算
  - 更高效的训练（小 batch 时效果优于 [CLIP](../models/CLIP.md)）

#### [ImageBind](../models/ImageBind.md) (2023)

- **论文**：[ImageBind: One Embedding Space To Bind Them All](https://arxiv.org/abs/2305.05665)
- **作者**：Rohit Girdhar et al. (Meta)
- **发表**：CVPR 2023
- **核心贡献**：
  - 将**6 种模态**（图像、文本、音频、深度、热成像、IMU）绑定到同一个嵌入空间
  - 仅需图像-文本对 + 图像-X 对的间接对齐（Emergent Alignment）
  - 零样本跨模态检索

### 核心贡献

1. **零样本迁移能力**：无需在目标任务上微调
2. **连接视觉与语言**：统一了 CV 和 NLP 的表征空间
3. **可扩展性**：预训练数据规模与性能正相关

### 技术要点

- **双塔架构 (Dual-Encoder)**：视觉塔和文本塔独立编码，通过对比损失对齐
- **对比损失函数设计**：
  - InfoNCE / NT-Xent 需要全局 softmax 归一化
  - [SigLIP](../models/SigLIP.md) 的 Sigmoid Loss 更高效
- **数据规模与质量的影响**：
  - 噪声数据可通过规模补偿（[ALIGN](../models/ALIGN.md)）
  - 数据清洗可提升效率（[CLIP](../models/CLIP.md) 的 WIT 数据集）

## 6.4 InfoNCE 损失的数学形式

标准的 InfoNCE 损失：

$$
\mathcal{L} = -\frac{1}{N} \sum_{i=1}^N \log \frac{\exp(q_i \cdot k_i^+ / \tau)}{\sum_{j=0}^K \exp(q_i \cdot k_j / \tau)}
$$

其中：

- $q_i$：查询（query），第 $i$ 个样本的编码
- $k_i^+$：正例键（positive key），$q_i$ 的增强视图
- $k_j$：负例键（negative key），其他样本的编码
- $K$：负例数量
- $\\tau$：温度系数

### 损失函数的变体

| 变体 | 公式特点 | 代表模型 |
|------|---------|---------|
| NT-Xent | InfoNCE 的对称化版本 | [SimCLR](../models/SimCLR.md) |
| NCE | 二分类（区分正负例） | [CPC](../models/CPC.md) 原始版 |
| Triplet Loss | 固定间隔的三元组 | [FaceNet](../models/FaceNet.md) |
| Hinge Loss | 基于间隔的对比 | 传统度量学习 |
| Sigmoid Loss | 独立二分类损失 | [SigLIP](../models/SigLIP.md) |
